# Thumbnail Captions

A thumbnail caption shown below thumbnail.

```jsx
const image1 = {
  src: "https://c2.staticflickr.com/9/8239/28897202241_1497bec71a_b.jpg",
  width: 248,
  height: 320,
  thumbnailCaption: "Big Ben",
}

return (
  <Gallery images={[image1]} />
);
```
